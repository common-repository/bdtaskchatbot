== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'bdtaskchatbot'
3. Activate bdtaskchatbot from your Plugins page.
4.Add this shortcode your theme header and Footer <?php echo do_shortcode('[bdtaskchatbot]') ;?>

= From WordPress.org =

1. Download bdtaskchatbot Plugin.
2. Upload the 'Bdtaskchatbot' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Bdtask 'Bdtaskchatbot' from your Plugins page.
4.Add this shortcode your theme header and Footer <?php echo do_shortcode('[bdtaskchatbot]') ;?>
